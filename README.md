# PUBG Teamer

**PUBG Teamer** is a simple Python program that helps you create random teams from a list of players in a PUBG lobby. It’s designed for small lobbies (4–8 players) and splits players into teams of appropriate sizes.

---

## Features

- Enter player names separated by commas.
- Automatically generates a random team based on the number of players:
  - 4 players → 2-player team
  - 5–6 players → 3-player team
  - 7–8 players → 4-player team
- Handles errors if the number of players is invalid.
- Written using Python functions for modularity.
